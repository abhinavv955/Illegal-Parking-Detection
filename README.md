# Illegal & Legal Parking Detection using YOLOvX

An AI-powered smart parking detection system developed using YOLOvX for detecting legal and illegal parking zones in real time with IP Camera Integration.

---

## Project Overview

This project was developed during a 10-day internship at St. John College of Engineering and Management.

The system uses Computer Vision and Deep Learning techniques to identify whether vehicles are parked in legal or illegal parking areas. The model performs real-time object detection and zone classification using YOLOvX.

The project also supports live monitoring through IP Camera Integration.

---

## Features

- Real-time vehicle detection
- Legal and illegal parking classification
- YOLOvX-based object detection
- IP Camera live stream integration
- Detection count display
- Smart parking monitoring system
- Android/mobile-friendly interface

## Project Workflow

1. Collect parking dataset images
2. Annotate legal and illegal parking zones
3. Upload dataset to Roboflow
4. Train YOLOvX model
5. Perform real-time detection
6. Display detection results on application
7. Integrate IP camera for live monitoring
